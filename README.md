## ✨ Key Features

### 🔧 Robust Engine
- **State Machine**: Three required states (Normal, Sick, Evolved) with visible transitions.
- **Persistence**: Save game progress to `localStorage` with offline decay calculations.
- **Event-Driven**: Clean architecture using a central Event Bus for UI updates.

### 🎨 Premium Aesthetics
- **CSS-Only Pet**: ChuChu is rendered entirely with CSS—no image assets required.
- **Dynamic Animations**: Breathing, tail-wagging, and state-specific overlays (Sick/Evolved).
- **Responsive Design**: Works perfectly on desktop and mobile browsers.

### 🥚 Personal Touches (Easter Eggs)
- **Nose Boop**: Click ChuChu's nose to make them sneeze!
- **Dizzy Spin**: Rapid clicks make ChuChu dizzy with spinning stars.
- **Konami Code**: Type the secret code for a rainbow-fabulous surprise.
- **Weekend Mode**: ChuChu wears a party hat on Saturdays and Sundays.
